🇬🇧 English Description

MapILS is a simple and lightweight interactive world map tool designed for flight simulation and aviation enthusiasts ✈️
It displays airports as green markers on a global map 🌍

By clicking on an airport, users can view:

Runway information 🛬
ILS (Instrument Landing System) approach points 📡
Approach angle and alignment data 📐

The goal of MapILS is to make IFR approaches and ILS understanding easier and more visual for pilots and simmers 🎯
